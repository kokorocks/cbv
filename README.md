# cbv

cbv stands for 'custom bible version,' we have 3 now

1. gtv (google translated version) (new testement only)
2. gzv (gen z version)
3. oev (old english version)

<hr>

gzv Bible

gen z version converts verses into the genz counterpart, without dumb words like 'skibidi'

<hr>

oev Bible

old english version converts it to how shakespeare would say back in his day
