# cbv

cbv stands for 'custom bible version,' we have 3 now

1. gtv (google translated version) (new testement only)
2. gzv (gen z version)
3. oev (old english version)

run and install
```bash
git clone https://github.com/kokorocks/cbv.git

cd cbv
```
\
and run with

```bash
python3 main.py
```

or on windows run

```bash
py main.py
```

<hr>

gtv Bible

just added support for the google translated version

example
```text
Matthew 1:1

Bible of the birth of Jesus Christ son of David son of Abraham.
```
<hr>

gzv Bible

gen z version converts verses into the genz counterpart, without dumb words like 'skibidi'

```text
Matthew 9:2

And, behold, they brought to him a man sick of the palsy, cap on a bed: and Jesus seeing their faith said unto the sick of the palsy; Son, be of valid cheer; thy sins be forgiven thee.
```

<hr>

oev Bible

old english version converts it to how shakespeare would say back in his day

```text
John 3:16

For God so lovedest thee world, thatth he gaveth his onlyest begotteneth Son, thatest whosoever believeth in him shouldest not perishest, but hathth everlasting lifeth.
```
